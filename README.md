# MIMIC-IV_DN_Mortality_Nomogram
There is a lack of prognostic tools for ICU patients with diabetic nephropathy (DN). Our aim was to develop and externally validate an interpretable, real-time risk prediction model. Using MIMIC IV v3.0 (with 4,478 ICU DN patients and a 7:3 split), we developed a LASSO Cox model to predict mortality within 365 days. We employed SHAP to improve interpretability and used segmented regression to analyse thresholds. For external validation (eICU CRD, n = 2,286; in-hospital mortality), we refitted a logistic model using the same predictors to align the outcomes between the two cohorts directly. The LASSO Cox model selected 14 predictors and achieved internal area under the curve (AUC) values of 0.816–0.844. SHAP ranked albumin, age and heart failure as the top contributors, and segmented regression showed no abrupt clinical thresholds. The sensitivity logistic model (MIMIC IV) yielded an AUC of 0.739 (95% CI 0.680–0.797). External validation (eICU CRD, n = 2,286) yielded an AUC of 0.784 (95% CI 0.732–0.837), a Brier score of 0.085, a calibration intercept of 0.476 (−0.207 to 1.199), a slope of 1.366 (1.023–1.753) and an HL p-value of 0.082. Decision curve analysis showed a net benefit across thresholds of 0.02–0.50. This validated, interpretable, dynamic nomogram provides good discrimination and fair calibration for ICU DN patients and is freely accessible as an interactive Shiny nomogram. Local intercept recalibration is recommended before bedside implementation.
